ColorSchemer
The New Chrome extension

It doesn't matter if you medically require a specific color scheme or if you're just a Pantone aficionado, ColorSchemer will change any site's color scheme for you. Web browsing is now more (or less) colorful.

Who can use the ColorSchemer?
  • Epileptics

  • The color blind

  • Anyone who uses the web
